# Instruction

1. Pull image:

`docker pull coldcloudgold/ma_lesson_10:otus`

2. Run container:

`docker run -p 8000:8000 ma_lesson_10:otus`

3. Check url:

`http://localhost:8000/health/`


| URL | Description |
|--- | --- |
| `http://localhost:8000/docs` | OpenAPI |
| `http://localhost:8000/health/` | Heath check |
